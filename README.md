## 基于人脸识别的登录以及合同在线签署系统

运用了serverless，人脸识别技术，redis以及pdfjs

前端使用vuejs，后端使用spring

## website

[点击访问](http://face.xinwuyun.cloud/#/login)

## 运行方法

```
git clone https://github.com/xinwuyun/facedec-SHOGOKI
yarn install
yarn serve
```

运行成功后访问 [http://localhost:8080/#/login](http://localhost:8080/#/login)

## 打包方法 

```
yarn build
```
